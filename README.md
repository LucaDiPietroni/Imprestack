# Service description

The application is designed to record which cultural works the user has experienced, 
capture the impressions they left, and plan what to explore next. 
The idea behind this project stemmed from a need to monitor the consumption of books, films, computer games etc. 
The project is built around a few simple principles: personal use only, 
local on-device database storage, and the ability to generate basic analytics on cultural engagement.

# Tech stack

- Kotlin
- Room
- Mockito
